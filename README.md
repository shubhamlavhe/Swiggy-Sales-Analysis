# Swiggy-Sales-Analysis
Excel Dashboard analyzing Swiggy sales trends across India.It provides insights into revenue trends, customer .The objective of this project is to analyze food delivery data and extract meaningful business insights, and regional performance.
Project Overview :
The objective of this project is to analyze food delivery data and extract meaningful business insights such as:
--Sales trends over time
--Customer ordering patterns
--Regional performance comparison
--Food category preferences

Key Insights
--Total sales reached ₹53M with consistent monthly growth
--Non-veg category dominates sales (~64%)
--Peak sales observed on weekends
--Top-performing cities include Bangalore, Hyderabad, and Mumbai
--Southern states contribute highest revenue
